Usage Guide


How to use Archa License in your project


Step 1: Add LICENSE file
~~~~~~~~~~~~~~~~~~~~~~~~
Copy the full license text into a file named ``LICENSE`` in the root of your repository.

You can find the full text here: :doc:`license`

Step 2: Update README
~~~~~~~~~~~~~~~~~~~~~
Add this badge to your README.md:

.. code-block:: markdown

   ## License
   This project is licensed under the Archa License - see the [LICENSE](LICENSE) file for details.

Step 3: Add Headers (Optional)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Add this to the top of your source files:

.. code-block:: text

   Copyright (c) 2026 Rogge Ramos
   Licensed under the Archa License

For GitHub Repository

1. Go to your repo on GitHub
2. Make sure ``LICENSE`` file exists in root
3. GitHub will auto-detect "Archa License"

Example Project Structure

::

   my-project/
   ├── LICENSE
   ├── README.md
   ├── docs/
   │   ├── conf.py
   │   ├── index.rst
   │   ├── introduction.rst
   │   ├── usage.rst
   │   └── license.rst
   └── src/
       └── ...

FAQ
---
**Q: Can I use it for commercial projects?**
Yes, 100% allowed.

**Q: Do I need to open source my code?**
No, unless you want to.

**Q: Can I modify the license?**
No, keep the original text. But you can add your name as licensor.
