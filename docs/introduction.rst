Introduction


What is Archa License?

**Archa License** is an open-source license created by Rogge Ramos.

The goal is simple: make licensing less confusing than MIT, Apache, GPL.

Why create Archa License?

* MIT is too short and vague
* GPL is too strict
* Apache is too long
* Notion docs glitch when you delete

Archa License = **Simple + Fair + No Glitch**.

Who can use it?

Anyone.

* Personal projects
* Commercial projects
* Open source libraries
* Startups

Philosophy

1.  **Freedom** — Use it anywhere
2.  **Credit** — Keep the author's name
3.  **No Evil** — Don't use it to harm

..

   Archa License is about trust, not legal jargon.
