What is Archa License?

Archa License is designed to be human-readable, developer-friendly, and free for everyone to use.

Features:

* Simple to understand
* Protects creators
* Allows commercial use
* Open source friendly

To use Archa License in your project:

1. Copy the ``LICENSE`` file to your repository
2. Add ``Archa License`` to your README
