What is Archa License?


**Archa License v1.0** is designed to be human-readable, developer-friendly, and free for everyone to use. 

It protects creators from Baguio to the world while keeping open source truly open.

Features
~~~~~~~~

* **Simple to understand** - No legal jargon, plain English
* **Protects creators** - Your work stays yours
* **Allows commercial use** - Use it in business, no problem
* **Open source friendly** - Compatible with Go, C, C++, Python, Ruby
* **Powered by Bot** - Auto-checks license with ``archalics-bot@v1``

Quick Start


To use Archa License in your project:

1. Copy the ``LICENSE`` file to your repository root
2. Add this badge to your ``README.md``:

.. code-block:: markdown

   [![Archa-1.0](https://img.shields.io/badge/License-Archa--1.0-green)](https://newssungoldentoday-dev.github.io/archalics/)

3. Install GitHub App:

.. code-block:: bash

   https://github.com/apps/archa-license-bot
