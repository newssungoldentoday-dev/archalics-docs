Report a Bug


Found a bug in Archa License text or in archalics tooling?

How to Report


1. Go to: https://github.com/newssungoldentoday-dev/archalics/issues
2. Click **New Issue** > **Bug Report**
3. Use this template:

.. code-block:: text

   Title: [BUG] Short description

   Description:
   What happened?

   Expected:
   What should have happened?

   License Text / File:
   Paste the section

   Environment:
   - OS: 
   - Version: v1.0

What NOT to report as bug

* Questions about interpretation -> use Discussions
* Commercial license request -> email licensor
* Your own project's bug -> not our repo

Response Time

We aim to triage within 72 hours.
